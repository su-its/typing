# Typing

静岡大学のITソルーション室が大学からの委託を受けて開発した、新入生向けタイピング練習ソフトウェアです。

## 概要

このプロジェクトはタイピング練習のためのウェブアプリケーションです。

## 主な機能

- タイピング練習
- ランキング表示
